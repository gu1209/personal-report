# Personal Report — Personal Portfolio Template

A bilingual (Chinese/English) personal portfolio template built with Next.js, deployable to GitHub Pages.

## Features
- Chinese/English toggle
- Responsive design
- PDF resume export
- GitHub Pages deployment via GitHub Actions

## Quick Start

1. Use the `Personal Report` tool to fill in your info and deploy automatically
2. Or clone this repo and edit `app/page.tsx` directly

## Development

```bash
npm install
npm run dev      # localhost:3000
npm run build    # build to ./out/
```
