# Resume File

Place your resume PDF file here and name it: resume.pdf

The portfolio will link to /resume.pdf for download.

To generate a PDF from your resume:
1. Prepare your resume in Word/Google Docs/LaTeX
2. Export as PDF
3. Save as this folder with filename: resume.pdf

Recommended specs:
- A4 or US Letter size
- Professional design
- Include all key experiences, education, skills, and certifications
- File size < 5MB
